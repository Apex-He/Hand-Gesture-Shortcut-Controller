# 基于手势识别的电脑快捷键控制器

本项目使用 OpenCV、MediaPipe 和 PyAutoGUI，通过摄像头捕捉手势来控制电脑系统的各种快捷操作。

## 🚀 支持的功能
- [0] 拳头：播放/暂停
- [1] 一根手指：降低音量
- [2] 两根手指：提高音量
- [3] 三根手指：PPT 下一页
- [4] 四根手指：PPT 上一页
- [5] 五根手指：新建浏览器标签
- [6] 六（大拇指+小指）：关闭浏览器标签
- [7] 七（三指捏合）：系统静音
- [8] 八（手枪）：向下翻页
- [9] 九（食指弯曲）：向上翻页
- [10] 十（食指中指交叉）：刷新页面

💡 **双手进阶机制：** 
除了上述单手传统手势外，本项目支持**双手手指个数累加**触发。
例如：当您不便做出复杂的单手“交叉指”时，可以直接伸出**双手各 5 根手指**（5 + 5 = 10）来触发网页刷新；或者伸出**一手 5 根 + 另一手 2 根**（5 + 2 = 7）来触发系统静音。这极大提升了日常使用的便利性与手势容错率。

## 🛠️ 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python gesture_controller.py
```
